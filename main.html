You can select elements using:

  • document.getElementById("id")
  • document.querySelector("selector")
  • document.querySelectorAll("selector")
  • document.getElementsByClassName("class")
  • document.getElementsByTagName("tag")

innerHTML: Allows inserting HTML content.
innerText: Displays only visible text.
textContent: Displays all text, including hidden elements.

Event delegation allows you to attach a single event listener to a parent element to manage events for its child elements dynamically.

Event bubbling is a propagation method where an event triggered on a child element bubbles up to its parent elements.

Creating an element: document.createElement("tag")
Adding to DOM: parent.appendChild(element)
Removing an element: element.remove()
